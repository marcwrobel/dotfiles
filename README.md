# dotfiles
Shell, git, etc config files.

## Installation and usage

    git clone https://github.com/marcwrobel/dotfiles.git ~/.config/dotfiles
    ~/.config/dotfiles/install

## Overview of the files

### Shell
* `.bashrc`
* `.bash_aliases`
* `.bash_functions`
* `.profile`

### Vim
* `.vimrc`

### Git
* `.gitconfig`
